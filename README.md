Developer Portfolio ⚡

A modern and responsive developer portfolio template built with React to showcase skills, projects, experience, and achievements.

✨ Features

Responsive and clean UI

Easy customization

GitHub project integration

Blog integration (Medium)

Social media links

Lottie animations support

📑 Sections

About Me

Skills

Education

Work Experience

Projects

Achievements & Certifications

Blogs

Contact Information

GitHub Profile

🛠 Technologies

React

GraphQL

Apollo Client

SCSS

Lottie Animations

⚙️ Getting Started
Clone the repository
cd developerFolio
npm install
npm start

The project will run at:

http://localhost:3000
✍️ Customization

Edit the file:

src/portfolio.js

Update your name, skills, projects, social links, and resume to personalize the portfolio.

🌐 Deployment

You can deploy the portfolio using:

GitHub Pages

Netlify
